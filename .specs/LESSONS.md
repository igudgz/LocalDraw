# LocalDraw - LESSONS

Playbook de licoes aprendidas do projeto, alimentado pelo Verifier (Review + QA) e consolidado pelo Metrics/PM Agent.

No TLC original, este arquivo e renderizado por `scripts/lessons.py` a partir de `lessons.json`. Esse automation ainda nao existe no LocalDraw, entao por enquanto este arquivo e mantido manualmente. Status da automacao: `TBD`.

Regras:

* So registrar licao a partir de falha fundamentada: mutant sobrevivente, gap de precisao de spec, criterio de aceite falho, ou desvio de spec (`SPEC_DEVIATION`).
* Um PASS limpo do Verifier nao gera licao.
* Licao deve ser acionavel e local ao projeto. Nada generico.
* Nao inventar licoes nem numeros.
* Licao confirmada deve ser carregada pelo PM Agent nas fases Specify e Design.

## Status

* confirmed: aplicar sempre.
* candidate: observado uma vez, ainda nao confirmado; nao bloquear decisao com base nele.
